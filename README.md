# #81 Habit Mirror

A hyper-minimalist daily habit tracker designed as a "mirror" to reflect your self-discipline. No streaks, no pressure, just a simple record of whether you kept your promise to yourself today.

## Core Philosophy
- **1 Tool / 1 Action**: Focus only on "Did I do it today?".
- **Habit Mirror**: No statistics or history. It only reflects your current state.
- **Auto-Reset**: The mirror clears every midnight, providing a fresh start daily.

## Features
- **Zero Friction**: Add a habit and tap to complete. That's it.
- **Privacy First**: No login, no tracking, no ads. All data stays in your browser's LocalStorage.
- **iOS Optimized**: Native-feel UI with specialized CSS for Safari/Mobile performance.
- **Responsive Design**: 15px rounded corners and minimalist monochrome aesthetic (#333 on #fff).

## How to Use
1. **Add**: Enter a habit you want to maintain (e.g., "Read 1 page," "Drink water").
2. **Reflect**: Tap the item to turn it black (Completed).
3. **Reset**: When the date changes, the list automatically unchecks itself for a new day.
4. **Remove**: Use the "×" button to delete habits you no longer need.

## Design System
- **Background**: `#ffffff`
- **Main Color**: `#333333`
- **Input Background**: `#fafafa`
- **Border Radius**: `15px` (Mandatory for all elements)
